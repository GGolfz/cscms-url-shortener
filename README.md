> A simple URL shortener build for aka.cscms.me powered by Azure Cosmos DB

## Prerequisite

- NodeJS
- Yarn
- Azure Cosmos DB

## Installation

Install all the dependency by using yarn.

```bash
yarn
```

Provide the environment variable in the .env file

> - COSMOSDB_ENDPOINT
> - COSMOSDB_KEY
> - PORT (Default to 3050)
> - ENTRYPOINT
